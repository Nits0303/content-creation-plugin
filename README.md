# Dynamic Content Creation Plugin

A Claude Cowork plugin for structured AI content generation across blogs, webpages, and social media.

## Features

- Structured intake system
- 43 content subtypes
- SEO-aware generation
- Social media content architecture
- Hallucination protection rules
- Sector-aware writing behavior

## Architecture

The system is based on a structured architecture covering:

- Blog content (12 subtypes)
- Web pages (16 subtypes)
- Social media content (15 subtypes)
